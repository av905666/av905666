# Sorting Visualizer

An interactive Sorting Visualizer built using HTML, CSS, and JavaScript to demonstrate how different sorting algorithms work through real-time animations.

## 🔗 Live Demo

[Click Here to View Project](https://av905666.github.io/av905666/)

## 🚀 Features

- Visual representation of sorting algorithms
- Random array generation
- Adjustable visualization speed
- Color-coded comparisons and swaps
- Responsive UI

## 📚 Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
